Get started Donate Examples repository
NodeJs ♥ Typescript
We control everything for you, making more easily your life and make your code more simple. You can inject classes, configure and destroy automatically, overriding classes to defineyour behavior.
Express ♥ Typescript
Yeah, express married with the Typescript! We can define rest endpoints more easily with the decorators @Get, @Post, @Put, @Del all provided by The Way.
Rest
We provided to you a loot of decorators to improve and make your code more clean. You can define a path, get body param, query param, path param, the headers incoming and more.
Http Server ♥ Swagger
We provide to you a server with express improved by best practices and the Swagger with the libraries swagger-jsdoc and swagger-ui-express
Yaml properties
You can use an application.properties.yml to disable or enable some features of this framework. You also can use this application properties in to you classes injecting the PropertiesConfiguration(@Inject())
